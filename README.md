# Python File Organizer

A Python automation script that organizes files into folders based on their file extensions.

## Features

- Automatically scans the current directory
- Creates folders if they don't exist
- Sorts files into categories
- Supports PDFs, Word files, Images, Videos, and Text files
- Beginner-friendly Python project

## Technologies Used

- Python 3
- os module
- shutil module

## Supported File Types

| Folder | Extensions |
|----------|------------|
| PDFs | .pdf |
| Word_Files | .docx, .doc |
| Images | .jpg, .jpeg, .png |
| Videos | .mp4, .mkv |
| Text_Files | .txt |

## How It Works

1. Scans the current working directory.
2. Creates category folders if they do not exist.
3. Identifies file extensions.
4. Moves files into their corresponding folders.
